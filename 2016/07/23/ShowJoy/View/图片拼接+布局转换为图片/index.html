发表于 2016-07-23   |   分类于 ShowJoy_View   |     |   次阅读   |   阅读次数

业务需求详细描述：最近产品说要在分享的商品图中添加一些其他图片和文字，然后拼接为一张图片，再分享到微信朋友圈，于是我就一脸懵逼了，但是没办法还是得做额！

然后整理了一下思路，主要有这么两条路线：

  1. 自己手动绘制。
  2. 将布局转换为图片。

很显然第一种方式是不合适的，无论是开发前还是开发后，成本都很大，所以果断选择了第二种方式。
